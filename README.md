安装

```
sudo apt-get install libnlopt-dev
或者从源码安装
cd ~
git clone https://github.com/stevengj/nlopt.git
cd nlopt
mkdir build
cd build
cmake ..
make
sudo make install

sudo apt-get install ros-noetic-trac-ik-lib

```
